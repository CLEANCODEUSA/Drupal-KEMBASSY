CONFIGURATION
-------------
1. Install module as usual. There is a default Developer API Key included at the following administration path. This can be changed:

   /admin/config/services/cms_custom_module_openweather

2. After Enabling the module, go to '/admin/structure/block' and place the open weather block in a region.

3. Test Weather Data by entering zip code at the following path: '/open/weather?zip_code=11211'
